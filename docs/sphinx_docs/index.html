ECOGEN’s documentation

Here is described the documentation of the ECOGEN open-source plateform dedicated to numerical simulation of compressible multiphase flows.

ECOGEN is an open-source project distributed under GNU GPLv3 License and available on GitHub.

It is developed by code-mphi members in partnership with AMU, Caltech and CNES.

In this document, ECOGEN’s user will find all necessary information to use the software.

Interested developers are encouraged to contribute in the project using the “pull request” feature on GitHub and can use the API documentation generated by Doxygen.
